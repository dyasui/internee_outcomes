IPUMS MLP LINKED CENSUS CROSSWALK FILE

This download contains an MLP crosswalk file in CSV format that identifies individuals across two censuses. The data also include a variable (STEP) that indicates the stage of the linking process in which the link was made. Users should create IPUMS data extracts for the appropriate census years and merge them onto the crosswalk.

For instructions, see the online user guide at https://usa.ipums.org/usa/mlp/mlp_crosswalk_guide.shtml.

For questions about IPUMS MLP, contact ipumsres@umn.edu.
