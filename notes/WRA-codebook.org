** Previous Address
<<prev_address>>
#+begin_quote
_Previous Address_
The code used will always be a 5 digit code which includes the two digit state code, the two digit county code, and the one digit size of city code.
Permanent address is the address to be coded if two are given.
...
The code for children born in Assembly Centers or Relocation centers will be 5 dashes...
...
#+end_quote

See [[file:../data/WRA_prev_address.csv]] for the full list of coded addresses that I have been able to determine.
There are several cases of towns of the same size category in the same city where I can't tell which is which.
In those cases, I have left the city name blank for in the row of that ~prev_address~ code.

_Instructions Used in coding Items on WRA 26 (Supplementary to Instruction to Coders)_
#+begin_quote
A previous address code was added for Alaska since approximately 150 Japanese were evacuated from there
#+end_quote

#+begin_quote
Where the previous address is outside the four evacuated states"
we have used the correct state code;
and have used the county code as given in the code book for all obtained from the Bureau of Agricultural Economis in Berkeley.
The size of city code is obtained by finding the population of the city from the First Series 1940 Census for the particular state.
#+end_quote
